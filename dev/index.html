PUPM.transfer_to_density — Method

function to convert Young's modulus to density E = E0(rho/rho0)^gamma

Example:

Enew = rand(Float64, 5)
E0 = 1.0
ρ0 = 1.0
γ = 1
ρ = transfer_to_density(Enew, E0, ρ0, γ)
